# 强化版Markdown清理插件
AstrBot插件：彻底移除机器人回复中的**、##、列表、引用等Markdown格式。

## 功能
- 移除所有`**`（加粗符号）；
- 移除`##`/`###`等标题符号；
- 移除`*`/`-`/`+`等列表符号；
- 移除`>`等引用符号。

## 安装方法
1. 打开AstrBot控制台 → 「插件」→ 「+安装」；
2. 选择「从Git仓库安装」，粘贴仓库链接：
   `https://github.com/youxin666/astrbot_plugin_markdown_cleaner_pro.git`；
3. 安装完成后启用插件。

## 使用说明
启用后，机器人回复会自动过滤Markdown格式，无需额外操作。
